## Description

Build the trestle-style base with mortise and tenon joinery.

### Steps
1. Cut mortises in legs
2. Cut tenons on stretchers and aprons
3. Dry fit and adjust
4. Glue up base assembly
5. Add through-wedged tenons for visual interest
