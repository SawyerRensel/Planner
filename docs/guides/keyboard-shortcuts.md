# Keyboard Shortcuts

Navigate Planner efficiently with keyboard shortcuts.

## Global Commands

Access via the Command Palette (++cmd+p++ / ++ctrl+p++):

| Command | Description |
|---------|-------------|
| `Planner: Open task list` | Open Task List view |
| `Planner: Open calendar` | Open Calendar view |
| `Planner: Open Timeline` | Open Timeline view |
| `Planner: Open Kanban` | Open Kanban view |
| `Planner: Create new item` | Open Item Modal |
| `Planner: Quick capture` | Open Item Modal |

## Calendar Navigation

| Command | Description |
|---------|-------------|
| `Planner: Calendar: Go to today` | Jump to today |
| `Planner: Calendar: Go to next period` | Move forward |
| `Planner: Calendar: Go to previous period` | Move backward |

!!! tip "Assign Hotkeys"
    Go to Settings > Hotkeys to assign keyboard shortcuts to these commands.

## Kanban Board

When the Kanban board is focused:

| Key | Action |
|-----|--------|
| ++arrow-up++ | Move focus to card above |
| ++arrow-down++ | Move focus to card below |
| ++arrow-left++ | Move focus to previous column |
| ++arrow-right++ | Move focus to next column |
| ++k++ | Move focus up (vim-style) |
| ++j++ | Move focus down (vim-style) |
| ++h++ | Move focus left (vim-style) |
| ++l++ | Move focus right (vim-style) |
| ++enter++ | Open focused card |
| ++space++ | Open focused card |
| ++escape++ | Clear focus |

### Focus Mode

1. Click the Kanban board to focus it
2. Use arrow keys or vim keys to navigate
3. Press Enter to open a card
4. Press Escape to exit focus mode

## Item Modal

| Key | Action |
|-----|--------|
| ++cmd+enter++ | Save and close |
| ++escape++ | Close without saving |
| ++tab++ | Move to next field |
| ++shift+tab++ | Move to previous field |

## Suggested Hotkeys

Consider assigning these hotkeys in Obsidian Settings:

| Hotkey | Command |
|--------|---------|
| ++cmd+shift+n++ | Create new item |
| ++cmd+shift+k++ | Open Kanban |
| ++cmd+shift+c++ | Open Calendar |
| ++cmd+shift+t++ | Open Task List |

## Mobile

On mobile devices:

- Tap to select items
- Long-press (200ms) to start dragging in Kanban
- Swipe gestures in Calendar view
