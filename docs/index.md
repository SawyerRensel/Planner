## Features

- **Built on Obsidian Bases** - Native integration with Obsidian's Bases feature
- **Multiple Views** - Calendar, Kanban, Timeline, and Task List views
- **Recurring Events** - Full RRule support for complex recurrence patterns
- **Drag & Drop** - Move tasks between columns, dates, and views
- **Mobile Support** - Touch-optimized with gesture support
- **Keyboard Navigation** - Full keyboard support with vim-style shortcuts
- **Customizable** - Configure colors, statuses, priorities, and more

## Quick Start

1. Install the plugin from Obsidian Community Plugins
2. Enable the plugin in Settings > Community Plugins
3. Click the ribbon icons to open your first view
4. Create your first item with the "Create new item" command
